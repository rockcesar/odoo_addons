First Module (for learning Odoo)

When install, add your user to First Module Group.

The following is assuming you have installed Odoo on http://localhost:8069.

For editing the module data, you have to enter to: http://localhost:8069/web, and access to the Config First Module menu.

To access to Person Website, the route is http://localhost:8069/person_website

You have here a module which has the basic development for Odoo. You are free to study and change the code.
